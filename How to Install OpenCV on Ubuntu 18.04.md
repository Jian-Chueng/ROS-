# How to Install OpenCV on Ubuntu 18.04

```shell
sudo apt update
sudo apt install python3-opencv
```

To verify the installation, import the cv2 module and print the OpenCV version:

```shell
python3 -c "import cv2; print(cv2.__version__)"
```
